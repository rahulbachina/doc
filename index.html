C) Getting Started with Ionic Framework - top

Ionic's ultimate goal is to make it easier to develop native mobile apps with HTML5, also known as Hybrid apps.

Install NodeJS

  • Install nodejs: http://nodejs.org/

Console Commands

  • $ npm install -g ionic cordova

Build and run app on iOS

  1. Install iOS emulator
  2. Change directory to app directory
  3. Add iOS platform
  4. Build app for iOS
  5. Emulate app
  6. Or run app on connected iOS device
  • $ sudo npm install -g ios-sim
  • $ ionic platform add ios
  • $ ionic build ios
  • $ ionic emulate ios
  • $ ionic run ios

E) Ionic Analytics - top

Why is analytics important?

How many users are on your app right now? How many of those will use your app tomorrow, or next week? Without information, you have no way of telling if your app is being used in the ways that you expect.If you've completed the Installation Guide, your Ionic app is already collecting data. This will get you core data that you need to understand your userbase, like retention and user growth rates. In addition, you get a view of what individual users do as they navigate your app.

E) Facebook Configuration - top

For use this application, you need to configure your app with Facebook and Parse.

Facebook : Acess Facebook Developper and create a app to http://developers.facebook.com

The Facebook Connect plugin to obtain access to the native FB application on iOS and Android. This plugin is not simple to install so make sure to check out the official docs.

ngCordova Full Documentation

Pre-Install

To use the FB plugin, you first have to create a new Facebook App inside of the Facebook developer portal at https://developers.facebook.com.

Retrieve the App ID and App Name, which will be required to setup your cordova app and for web-development.

Insert Config Keys in www/js/facebook.config

iOS Install

Download the repo from https://github.com/Wizcorp/phonegap-facebook-plugin. Then type in the following commands in your Terminal.

$ cordova plugin add https://github.com/Wizcorp/phonegap-facebook-plugin.git --variable APP_ID="123456789" --variable APP_NAME="myApplication"

F) Parse Getting Stared - top

For use this application, you need to configure your app with Parse.

Parse : Parse is a BaaS ( Backend as a Service ), wich allows you to manage your application ( users, messages, ... ), ans also to be able to send and receive messages. Create a free account to http://parse.com, and create a app. Get the AppKeys ( Application ID & Client Key ), and add to your project ( www/js/config.js ).

Access Parse Site

Create Account

Click for Data Option

Click for Web Option

Click for New Project Option

Copy Javascripts Keys and modify config file in www/js/config.js

Enter in Settings for Show Keys

Insert Config Keys in www/js/parse.config.js
